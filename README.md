#  Hallucination Hunter

A interactive web-based game designed to raise awareness about AI hallucinations. Users are presented with two statements—one factual and one "hallucinated"—and must determine which is which.

##  How it Works
1. **The Challenge:** The game displays two cards. One contains a verified fact, and the other contains a subtle or blatant AI-style hallucination.
2. **The Goal:** Pick the real information to increase your score.
3. **The Reveal:** After choosing, the game highlights exactly which part of the text was a hallucination and explains why it's incorrect.

##  Tech Stack
- **HTML5** for structure.
- **CSS3** for a dark-mode, responsive UI.
- **JavaScript (Vanilla)** for game logic and DOM manipulation.

##  Project Structure
- `index.html`: The entire application (HTML, CSS, and JS) is contained in this single file for easy deployment via GitHub Pages.
